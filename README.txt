Adds an area to suggest another product via shortcode on product page

== Description ==

This plugin adds an area to suggest another alternative product via shortcode on a product page.  It displays the products brand, price difference in percentage, thumbnail of product, product title, and price in a stylized box.  Add the shortcode to your product page template to display dynamically, enter the specific details through the admin panel by editing the product.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `another-product.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
